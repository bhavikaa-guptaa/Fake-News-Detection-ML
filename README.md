# Fake News Detection using Machine Learning

## Problem Statement
The spread of fake news on digital platforms can mislead users and create misinformation.

## Solution
This project uses a machine learning model to classify news articles as real or fake
based on textual features.

## Tech Stack
- Python
- Scikit-learn
- TF-IDF Vectorization
- Logistic Regression

## Features
- Text-based news classification
- Simple command-line interface
- Lightweight ML pipeline

## How to Run
pip install -r requirements.txt  
python train_model.py  
python predict.py
